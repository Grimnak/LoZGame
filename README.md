## Controls:
W, A, S, D, Arrow Keys - Controls the Player and Inventory Select

Z - Attack buttons for Player

N - Use item button

I - Open Inventory

Enter - Selects Item in Inventory

Q - Exits the game

R - Resets the game to its default state

Mouse - Moves rooms in the direction you click.
